# **Vim**
​
-   vi 호환 텍스트 편집기
​<br>
​<br>

## Vim command
​
```bash
$ vi readme.txt #vi로 readme.txt를 연다
```
​
```bash
i   #normal mode  진입
esc #insert mode  진입
:   #command mode 진입, 저장할 때는 command mode를 사용한다
```
​<br>
​<br>
​
## Command mode
​
```bash
:q   #나간다
:q!  #저장 안하고 나간다
:w   #write, 저장을 의미한다
:wq  #write and quit, 저장을 하고 나간다
```
