# **Filter**

```css
.container {
  filter: blur();
  filter: brightness();
  filter: contrast();
  filter: drop-shadow();
  filter: grayscale();
  filter: hue-rotate();
  filter: invert();
  filter: opacity();
  filter: saturate();
  filter: sepia();
}
```
